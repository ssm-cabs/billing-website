# billing-website

Billing and ride entry web application for SSM Cabs. This app is used by office staff
to enter daily ride data, manage pricing per corporate company, and generate monthly
billing and invoices.

## Tech Stack

- Next.js (App Router) with JavaScript
- Google Sheets + Apps Script backend (planned)
- GitHub Pages for hosting (planned)

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
